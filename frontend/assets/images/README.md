# Images des logos sociaux

Placez les logos officiels des réseaux sociaux dans ce dossier :

- `google_logo.png` - Logo Google (G multicolore)
- `apple_logo.png` - Logo Apple (pomme noire)
- `microsoft_logo.png` - Logo Microsoft (4 carrés colorés)

## Spécifications recommandées

- **Format** : PNG avec transparence
- **Taille** : 24x24 pixels minimum (ou plus grand, sera redimensionné automatiquement)
- **Couleurs** : Utilisez les logos officiels avec leurs couleurs de marque

## Où trouver les logos officiels

- **Google** : https://developers.google.com/identity/branding-guidelines
- **Apple** : https://developer.apple.com/design/resources/
- **Microsoft** : https://www.microsoft.com/en-us/legal/intellectualproperty/trademarks/usage/general

## Note

Si les images ne sont pas trouvées, des icônes de fallback seront utilisées automatiquement.
